- I haven't really explained that domain conditions used to check if terms are well-formed in a proposition must require you to read the proposition left to right. For example, if $P\equiv x\ne 0\land 1/x = 2$, the fact that $1/x$ is well-formed comes from the first clause $x\ne 0$. If $x\ne 0$ is true, we move on to checking whether $1/x = 2$ and then determine the truth of $P$. Otherwise, if $x\ne 0$ is false, we stop there before looking at $1/x = 2$ and conclude $P$ is false. This has the consequence that logical connectives that are usually symmetric are not symmetric here: $A\land B$ is not treated the same as $B\land A$, because e.g. $x\ne 0\land 1/x = 2$ is a true/false well-formed formula, but $1/x = 2\land x\ne 0$ is not. The same for quantifiers: $\exists x\in\mathbb{R}\setminus\{0\}\, (1/x > 0)$ is ok but $\exists x\in\mathbb{R}\, (1/x > 0)$ is not.
